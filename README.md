# 项目简介
产品名称：请教一下
定位：年轻人的付费咨询产品（微信小程序）
减少年轻人由于信息不对称产生的迷茫和焦虑。例如，咨询高考志愿填报，大学专业选择，大学生活规划，大学生择业等。

## 预期功能

- [x] 用户登陆
- [x] 付费提问
- [x] 展示本人主页
- [x] 搜索回答
- [x] 回答
- [x] 付费旁听
- [x] 评价订单
- [x] 查看问答详情
- [x] 查看历史订单
  * 未支付
  * 未评价
  * 已完成
- [x] 查看答主详情
  * 用户评价
  * 公开咨询
- [ ] 统一UI风格
- [ ] 增加新功能
  * 点赞关注
  * 限时抢购
- [ ] 小程序上线
- [ ] 接入微信支付API
